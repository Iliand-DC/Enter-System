Это один из самых простых проектов, с которого можно начать изучение файловых систем на C++. 
Проект включает в себя процесс регистрации пользователя путем запроса имени пользователя и пароля. 
После успешной регистрации создается пользовательский файл с учетными данными.

Если пользователя не существует, при входе в систему будет показана ошибка. 
Вы также узнаете, как использовать Visual Studio для создания простого проекта.
